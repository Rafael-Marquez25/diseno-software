\doxysubsection{Detailed Description}
Clase principal del sistema de gestión de citas. 

La clase \doxylink{classSystem}{System} implementa el patrón Singleton y se encarga de\+:
\begin{DoxyItemize}
\item Gestionar la conexión con la base de datos SQLite.
\item Controlar el inicio de sesión y registro de usuarios.
\item Crear usuarios según su rol (\doxylink{classAdmin}{Admin}, \doxylink{classMedic}{Medic}, \doxylink{classPatient}{Patient}).
\item Administrar robots disponibles para acompañamiento.
\item Gestionar médicos y sus horarios. 
\end{DoxyItemize}
